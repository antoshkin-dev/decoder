# История изменений в проекте Decoder
## Актуальные нововведения
- Для понимания работы декодера, добавлен модуль swagger в виде нового контейнера. Обращаться к Swagger следует через url %https://decoder.server%/swagger.
Для его настройки следует указать реальный url до вашего сервера декодера в файл `./swagger/openapi.json`:
```
servers:
  - url: 'https://decoder.qwer.kz:8443/api'
```
