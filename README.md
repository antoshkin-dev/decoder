# Decoder
## О проекте
Веб сервис, обеспечивающий шифрование модулей приложения WebApps
## Структура каталогов и их назначение
`decoderapp` Каталог исходных файлов проекта, с которым работает контейнер PHP  
`nginx` Каталог инициализации контейнера веб-сервера NGINX  
`├─ certs` Каталог, в котором следует разместить сертификаты веб-сервера  
`├─ conf.d` Каталог конфигурация NGINX  
`└─ logs` Журналы /var/log  
`php` Каталог инициализации контейнера PHPFPM  
`├─ ./Dockerfile` Данный контейнер собирается с дополнительными параметрами  
`├─ ./custom.ini` Дополнительные настройки PHP  
`├─ ./init.sh` Измененная точка запуска контейнера  
`└─ logs` Журналы /var/log  
`./.env-common` Файл, который отсутствует на GIT, но должен быть заполнен для запуска кучи  
`./docker-compose.yml` Файл-кучи  

## Установка
1. Разместить сертификаты веб-сервера и скорректировать пути в файле конфигурации NGINX;
2. Скорректировать параметры в файле-куче `./docker-compose.yml`:
    + сетевые адреса и порты;
    + часовой пояс;
3. Внести изменения в файл-настроек PHP `php/custom.ini`
4. Запустить сборку кучи: `docker compose up --build -d`
